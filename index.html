Overview

CloudMRHub provides an integrated platform for developing applications, computational pipelines, and web solutions for medical imaging and radiomics. Our ecosystem supports both Kubernetes-based implementations and serverless architectures, ensuring scalability and maintainability.

Applications

  • MR Optimum: A web application for Signal-to-Noise Ratio (SNR) calculations. Supports raw MR data compatible with ISMRMRD format.
    • Python Package: mroptimum-tools
    • Frontend: React Application
    • Backend: Serverless Implementation
  • CAMRIE: Simulates virtual MRI experiments using Bloch simulation.
    • Frontend: React Application
    • LambdaKoma: Cloud-based Bloch simulator for v2.
  • RADO: Radiomics tool for feature extraction.
    • Features Extractor: ITK-based library developed during BD2Decide.
    • Backend: Serverless Implementation
  • TESS: Tool for temperature analysis in MRI workflows.
    • Python Package: tess
    • Frontend: React Application
  • Poirot: Performance analysis tool.
    • Frontend: React Application
    • Python Package: PerformanceMaster for backend operations.

Shared Tools

  • cloudmrhub: Python library for MRI computations and reconstructions.
  • cmrawspy: Python package for handling CloudMR data.
  • cloudmr-ux: UI library for frontend components.
  • ComputingEngine: Foundational backend for Cloud MR applications.
  • Raider: Python library for handling Siemens DAT files.
  • pynico: Utility for string and file management.
  • registrationMplus: Advanced image registration algorithms.
  • pydaug: Data augmentation library for medical imaging.
